# Repository Layout

3 npm workspaces (`sdk` / `core` / `example`; the only dependency direction is `example → sdk ← core`) plus a bundled deployer (ADR 0015).

```
sdk/src/            contract layer (@eris/sdk)
  actionSchema.ts     zod schema for agent actions (source of the <schema> generated for prompt agents)
  protocols/          protocol adapters (uniswap / balancer / curve / aave / gmx + registry / oracles)
  chain.ts            clients, anvil cheatcodes, resetFork (snapshot/revert on local)
  observation.ts      observationFor (the environment's scoring and the agent's observation use the same one)
  runConfig.ts        YAML config schema (nested lowercase → internal keys)
  constants*.ts       venue addresses (fork: constants.ts / local: constants.local.ts = generated by gen:local-constants)
core/src/           environment daemon + scoring (participants do not touch this)
  cli/                CLI entries (anvil / sim-realtime / backtest / checkOrdering)
  realtime/           coordinator / priceFeed / events(stress) / vulnEvents / reconstruct / agentProcess / flowProcess
  flow/               orderflow bot (logic.ts = pure functions / market-maker.ts = independent process)
  backtest/           backtest shared helpers (state dump manifest / fingerprint / regime resolution; ADR 0016)
  postRunCheck.ts     post-run rule checks (fee limit overruns → violations)
example/agents/     participant template (1 agent = 1 directory is the unit of copy and submission)
  runtime/            generic driver scripts (bot / read / send / llm / prompt / agentLog; reserved names)
  lib/                shared strategy helpers (markets.ts, etc.; reserved name)
  <id>/               the agent itself (agent.ts's decide/run, or prompt.md)
contracts/          PriceFeed + mock oracles + FlashArb (Foundry)
deployer/           bundled deploy orchestrator (self-contained subpackage that deploys all 5 venues to an empty anvil)
config/             YAML config (example.yaml = template / vuln-test.yaml / regimes/ = official regimes; ADR 0016)
backtest/state/     state dump + manifest (gen:state-dump output; gitignored)
docs/guide/         usage guides (this directory)
docs/adr/           architecture decision records (ADR 0001–0016)
scripts/            gen:local-constants / gen:state-dump / check:strategy / check:boundaries / bundle:agent
test/               unit tests (node --test)
runs/               run output (summary.json / events.jsonl / blocks.csv / agents/<id>.jsonl)
```
